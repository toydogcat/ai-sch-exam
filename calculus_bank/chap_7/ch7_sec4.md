[q:1, type:single, ans:C]
求 $\int \frac{2x + 2}{x^2 + 2x} dx$ 的最一般不定積分。
(A) $2\ln|x| + C$
(B) $\frac{1}{2}\ln|x^2 + 2x| + C$
(C) $\ln|x^2 + 2x| + C$
(D) $\ln|x + 2| + C$
<!-- solution -->
令 $u = x^2 + 2x \implies du = (2x + 2)dx$。故積分為 $\int \frac{1}{u} du = \ln|u| + C = \ln|x^2 + 2x| + C$。

(A) $\frac{1}{3}(\ln|x| - \ln|x + 3|) + C$
(B) $\ln|x| - \ln|x + 3| + C$
