# 微積分題庫 - 3.2 節：均值定理

本節收錄 3.2 節相關題目。

---

[q:1, type:single, ans:C]
已知 $f(x) = x^2 - 3x$ 在 $[0, 3]$ 上滿足 Rolle 定理，求對應的 $c$ 值為何？
(A) $3$
(B) $0$
(C) $1.5$
(D) $1$
<!-- solution -->
$f'(c) = 2c - 3 = 0 \implies c = 1.5$。

---

[q:2, type:single, ans:A]
下列何者為 Rolle 定理與均值定理不需滿足的條件？
(A) 函數值恆為正
(B) 在開區間內可導
(C) 在閉區間內連續
(D) 定義域長度不為 0
<!-- solution -->
Rolle 定理與均值定理都不要求函數值恆為正。

[q:3, type:single, ans:A]
已知 $f(x) = x^2$ 在閉區間 $[0, 5]$ 上滿足均值定理，試問對應的 $c$ 值為何？
(A) $2.5$
(B) $5$
(C) $1$
(D) $10$
<!-- solution -->
$f'(c) = 2c = \frac{f(5) - f(0)}{5 - 0} = \frac{25}{5} = 5 \implies c = 2.5$。
