[q:1, type:single, ans:A]
求 $f(x) = \cos x$ 的最一般反導函數。
(A) $\sin x + C$
(B) $-\sin x + C$
(C) $\cos x + C$
(D) $-\cos x + C$
<!-- solution -->
$(\sin x)' = \cos x \implies \sin x + C$。
