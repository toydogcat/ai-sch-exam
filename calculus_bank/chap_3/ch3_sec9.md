# 微積分題庫 - 3.9 節：反導函數

本節收錄 3.9 節相關題目。

---

[q:1, type:single, ans:A]
求 $f(x) = \cos x$ 的最一般反導函數為何？
(A) $\sin x + C$
(B) $-\sin x + C$
(C) $\cos x + C$
(D) $-\cos x + C$
<!-- solution -->
因為 $(\sin x + C)' = \cos x$，故反導函數為 $\sin x + C$。

---

(A) $\frac{4}{2}x^2 + C$
(B) $8x + C$
(C) $4x^3 + C$
(D) $\frac{4}{3}x^3 + C$
<!-- solution -->
