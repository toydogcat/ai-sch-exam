# 微積分題庫 - 第 3 章：綜合題目

本節為第 3 章綜合題目。

---

[q:1, type:single, ans:B]
【綜合】求極限值 $\lim_{x \to \infty} \frac{2x^2 + 1}{x^2 + 2}$。
(A) $1$
(B) $2$
(C) $2$
(D) $\infty$
<!-- solution -->
最高次方係數比值為 $2$。

---

[q:2, type:single, ans:C]
【綜合】求 $f(x) = \cos x$ 的最一般反導函數。
(A) $\cos x + C$
(B) $-\sin x + C$
(C) $\sin x + C$
(D) $-\cos x + C$
<!-- solution -->
積分得 $\sin x + C$。

---

[q:3, type:single, ans:B]
【綜合】求函數 $f(x) = x^2 - 8x$ 之臨界點為何？
(A) $-4$
(B) $4$
(C) $0$
(D) $1$
<!-- solution -->
$f'(x) = 2x - 8 = 0 \implies x = 4$。
