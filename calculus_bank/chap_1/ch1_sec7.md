# 微積分題庫 - 1.7 節：極限的精確定義

本節收錄 1.7 節相關題目。

[q:1, type:single, ans:B]
極限精確定義中，若 $|x - a| < \delta$，則何者成立？
(A) $|f(x) - a| < \epsilon$
(B) $|f(x) - L| < \epsilon$
(C) $|x - L| < \epsilon$
(D) $|f(x) - L| > \epsilon$
<!-- solution -->
根據極限定義，當 $x$ 趨近於 $a$（$|x - a| < \delta$）時，函數值趨近於 $L$（$|f(x) - L| < \epsilon$）。

---

[q:2, type:single, ans:B]
設 $\lim_{x \to 1} (4x - 1) = 3$，若 $\epsilon = 0.1$，試求最大的 $\delta$ 滿足定義。
(A) $0.1$
(B) $0.025$
(C) $0.05$
(D) $0.2$
<!-- solution -->
由 $|(4x-1) - (3)| < 0.1 \implies 4|x-1| < 0.1 \implies |x-1| < \frac{0.1}{4}$。故 $\delta = 0.025$。
