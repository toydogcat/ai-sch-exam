# 微積分題庫 - 2.3 節：微分公式

本節收錄 2.3 節相關題目。

[q:1, type:single, ans:A]
求函數 $f(x) = (2x+1)(x+2)$ 的導函數 $f'(x)$。
(A) $4x + 5$
(B) $2x + 3$
(C) $2x^2 + 1$
(D) $x + 2$
<!-- solution -->
展開得 $f(x) = 2x^2 + 5x + 2$，故微分為 $4x + 5$。
