# 微積分題庫 - 6.6 節：反三角函數

本節收錄 6.6 節相關題目。

---

[q:1, type:single, ans:C]
求 $\int \frac{1}{1 + x^2} dx$ 的最一般不定積分。
(A) $\cos^{-1} x + C$
(B) $\sin^{-1} x + C$
(C) $\tan^{-1} x + C$
(D) $\sec^{-1} x + C$
<!-- solution -->
對反三角函數求導公式反向積分：$\int \frac{1}{1 + x^2} dx = \tan^{-1} x + C$。

---

[q:2, type:single, ans:A]
求反三角函數 $y = \tan^{-1}(x)$ 的一階導數 $\frac{dy}{dx}$。
(A) $\frac{1}{1 + x^2}$
(B) $\frac{1}{\sqrt{1 - x^2}}$
(C) $\frac{1}{x^2 - 1}$
(D) $\frac{1}{1 - x^2}$
<!-- solution -->
反正切函數的導數公式為 $\frac{d}{dx} \tan^{-1} x = \frac{1}{1 + x^2}$。
