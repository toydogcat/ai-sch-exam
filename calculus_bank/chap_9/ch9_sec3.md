[q:1, type:single, ans:B]
對於 $\frac{dy}{dx} = \frac{y}{x}$，其最一般通解為何？
(A) $y = C e^x$
(B) $y = C x$
(C) $y = x + C$
(D) $y = C x^2$
<!-- solution -->
分離變數得 $\int \frac{1}{y} dy = \int \frac{1}{x} dx \implies \ln|y| = \ln|x| + C_1 \implies y = C x$。
