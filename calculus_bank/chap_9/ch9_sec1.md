[q:1, type:single, ans:C]
判斷下列哪一個函數是 $\frac{d^2y}{dx^2} + 4y = 0$ 的解？
(A) $e^{2x}$
(B) $\sin(2x)$
(C) $\cos(2x)$
(D) $x^{2}$
<!-- solution -->
對 $y = \cos(2x)$ 求二次導數得 $y'' = -4\cos(2x)$。代入原微分方程滿足 $y'' + 4y = 0$。

---

[q:2, type:single, ans:B]
對於一階常微分方程 $\frac{dy}{dt} = 3y$，其最一般的通解形式為何？
(A) $y(t) = C e^{-3t}$
(B) $y(t) = C e^{3t}$
(C) $y(t) = 3t + C$
(D) $y(t) = C e^{t/3}$
<!-- solution -->
分離變數積分：$\int \frac{1}{y} dy = \int a dt \implies \ln|y| = at + C_1 \implies y(t) = C e^{at}$。

<!-- solution -->
積分為 $y(x) = \frac{3}{2}x^2 + C$。代入 $y(0)=5 \implies C=5$。故 $y(x) = \frac{3}{2}x^2 + 5$。代入 $x=2 \implies y(2) = \frac{3}{2}(4) + 5 = 23 + 5$。
